# Telegram Savings Ecosystem (SLH / SELA)

מערכת חיסכון, השקעות וקהילה על גבי טלגרם, המשלבת:

- מודלים מתמטיים מתקדמים (פיבונאצ'י, צמיחה אקספוננציאלית, S‑curve, אפקט רשת)
- מודל תשואות מדורג לפי ותק ומעמד (pioneer / early / community / standard)
- ניהול קהילה, קרן קהילה, שקיפות מלאה ותחקור השפעה חברתית
- תשלומי הצטרפות (למשל 39 ₪) והנפקת מטבעות SLH ללקוחות
- אינטגרציה לבלוקצ'יין (BSC / Web3) וחוזים חכמים תומכי טיירים והפניות
- מערכת ניהול סיכונים, דיווח רגולטורי ואבטחה ברמת FinTech

ה‑repo הזה יושב מעל שער ה‑API / בוט שלך (`botshop` / `slhnet-telegram-gateway`) ונותן שכבת מושג, אדריכלות ותיעוד.
